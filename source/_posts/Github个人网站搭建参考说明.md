# Github + 个人网站搭建参考说明

个人网站是依托Github Page实现的，不用花钱。你可以将你学的内容同步到网站，或者发表你的思考。形成自己的知识库的同时，我们可以很轻松的互相学到新知识，知道大家都做了什么事，方便我们之间的交流。并且思源有插件可以一键发布,配合picgo图床,可以很方便地发表自己的文章。

注意:

1. 如果想用思源的一键发表，就不能在本地用node.js编译，而要使用github action
2. 思源集市中的picgo插件有问题，不能使用内置的picgo服务器，必须要安装对应软件，使用api上传图片
3. HEXO有很多好看的主题可以自行配置，参考：[思源笔记配合插件实现HEXO文章快速发布 - Shepherd010](https://shepherd010.github.io/post/siyuan-notes-and-plugin-realize-the-rapid-release-of-hexo-articles-za8u7r.html)
4. 上传图片需要有对应的云服务支持，我使用的是阿里云，同样地，你也可以开通思源的终身会员使用阿里云同步信息，我的文章里有写，如果发现不全面，可以参考：
